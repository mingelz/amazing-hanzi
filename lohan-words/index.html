罗汉字

在汉字中，有不少都是由三个已有的字叠在一起组成了一个新的汉字，我形象的称之为「罗汉字」。

说明

  • 所列汉字按笔划排序；
  • 各拼音后边的数字表示读音的声调；
  • 感谢 极点五笔，在查询各字的读音及解释时使用了极点五笔查询插件1.1版；
  • 如果哪位大侠还知道这里没有列出来的罗汉字，欢迎提 issue。

补充说明：

这是我在上高中时，与坐我后排的加菲同学一起研究的产物。后来上大学后，先是放到我的blogcn博客上。回想当时，我们抱着加菲那本 50 年代产的字典翻啊翻...高中生活虽紧张，但也有趣。

以上这些都是可以在 GB2312 字符集中查找到的字符。另外，我国还有一个完美接入 Unicode 编码的 GB18030 大字符集标准，其中包括更多的「罗汉字」。不过那些都属于生僻字，用于特殊领域（如人名、古籍的录入等），很多读音和字义已不可考。比如（后面的编码为该字的内码）：

  • 三个「市」：d848dd48
  • 三个「兔」：d841dcd7
  • 三个「光」：d841dcc9
  • 三个「少」：d846df8f
  • 三个「可」：d843dfb3
  • 三个「面」：d864de56

这个 GB18030 标准只支持 Windows 2000 以上的系统，但 Windows 2000 系统需要下载「Surrogate更新」和「GB18030 Support Package简体中文版」补丁包才可以对其支持。具体的下载地址可登录 微软中国网站 用以上关键字搜索得到。

而 Windows XP 已经将其集成，可以直接使用。调用方法：在输入法图标上点击鼠标右键，选择「设置」打开「文字服务和输入语言」对话框，其中列出了已经安装的输入法。点击右侧的「添加(D)...」按钮打开「添加输入语言」对话框，在「键盘布局/输入法」中选择「中文(简体) - 内码」，然后确定添加即可。

该「中文(简体) - 内码」输入法的图标为「区」字，也就是说它其实是采取的区位输入方式。如果要输入 GB18030 字符集中的汉字，在调出「中文(简体) - 内码」后，点击输入法状态栏的「区位」使之成为「UNICODE」，然后就可以输出 GB18030 字符集的汉字了。

另外，微软拼音 2003 也有内码输入方式：打开输入法功能菜单，选择「内码输入 - Unicode码输入」，输入法状态条出现「UNICODE」标志即可。

注意：

  1. 必须使用支持大字符集的字体，如「宋体 - 18030」；
  2. Unicode 码必须使用英文小写字母和主键盘数字键输入，大写字母和小键盘无效；
  3. 在录入 8 位 Unicode 码时，字符可能一时显示不出来，不必管它，直接输入全部 Unicode 码，相应字符即可出现。

以上「补充说明」参考自 《生僻字输入速查手册》（ISBN 7-5323-7955-8），上海科学技术出版社，唐素荣、沈惠璋等编著，2005年3月第1版。如果你在这方面有需求，可以去买一本，450+页，售价35元。
